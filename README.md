# Словце

Игра **Словце** является клоном игры в слова **Wordle**
(https://www.nytimes.com/games/wordle/index.html).

## Правила

Игрок должен угадать загаданное слово из заданного кол-ва букв (обычно пяти).  
На разгадывание слова у него несколько попыток (обычно шесть).

## Особенности

Данная реализация имеет консольный интерфейс.  
Слова для игры взяты из толкового словаря Ожегова.
